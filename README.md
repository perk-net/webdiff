# 产品介绍

**网页侦探（WebDiff）** 是一款 **多功能定时任务监控工具**，可用来监控 **网页、接口、RSS、域名与证书、Ping 连通**，还可以通过 **自定义脚本** 扩展更多场景，在触发条件满足时通过微信、钉钉、飞书等渠道第一时间通知你。提供 **桌面客户端**（Windows / macOS / Linux）与 **Web 部署**（Linux安装包 / Docker部署）两种使用方式。

## 它能做什么？

- 🌐 **网站内容**：真实浏览器加载页面，可视化点选元素，内容变化提醒
- 🔌 **HTTP 请求**：定期巡检接口，按响应码、响应体或内容变化告警
- 📡 **RSS 订阅**：跟踪博客、播客等源的新条目与字段变化
- 📅 **域名 / 证书**：域名 与 SSL 临期提醒，运维必备
- 📶 **Ping 检测**：监控主机连通性与平均延迟
- 📜 **自定义脚本**：在运行节点执行 JavaScript / TypeScript / Shell / Python 脚本，按输出与退出码告警
- ⏱ **灵活调度**：从每 1 分钟到自定义 Cron，各类任务共用
- 🔔 **多渠道通知**：一任务可绑多个渠道，变化即时推送
- 📜 **完整追溯**：执行记录、差异对比与通知回执均可回查

## 适合谁用？

| 角色 | 典型场景 |
| --- | --- |
| 普通用户 | 商品比价、录取公示、活动报名、票务抢购 |
| 行业从业者 | 招投标公告、政策文件、行业研报 |
| 运维 / 开发 | 接口健康检查、Ping 巡检、证书与域名到期、自定义脚本巡检 |
| 团队 / 运营 | 竞品官网、RSS 资讯、活动数据看板 |

## 为什么选网页侦探？

### 1. 多类型任务，统一管理

网页元素、HTTP 接口、RSS、Ping、域名与证书、自定义脚本在同一界面管理，不必为不同场景切换多个工具。

### 2. 真实浏览器 + 可视化挑选

网站类任务用浏览器完整渲染页面，**鼠标点哪监控哪**，自动生成 XPath / CSS，零代码上手。

### 3. 本地优先 + 灵活部署

- 提供桌面客户端与 Web 部署，功能完全一致，可按场景自由选择
- 任务、Cookie、执行快照默认保存在本机
- 微信扫码登录，会员权益云端同步
- 通过 **cookie plus** 同步浏览器登录态，后台页面也能监控

### 4. 通知渠道丰富
 
- pushplus（微信、短信、语音、App等）
- 本地桌面/浏览器通知
- 钉钉机器人
- 企业微信机器人
- 飞书机器人
- 邮件
- 自定义 Webhook
