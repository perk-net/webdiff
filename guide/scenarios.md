# 常见使用场景

以下按 **任务类型** 归纳典型用法，可按需组合多种任务盯同一业务。

## 🌐 网站内容监控

### 电商比价 / 抢购

- **元素**：价格、原价、库存、优惠券按钮
- **规则**：内容变化、数值比较（低于某价）
- **频率**：日常 30 分钟～1 小时；抢购前可提高到 1～10 分钟（视会员等级）
- **通知**：pushplus + 本地通知

### 学校公告 / 录取查询

- **元素**：公告列表第一条标题 + 时间，或附件链接
- **频率**：每 30 分钟
- **通知**：pushplus

### 招投标 / 政策公示

- **元素**：列表首条标题 + 发布时间
- **频率**：每 10～30 分钟
- **通知**：钉钉 / 企业微信群机器人

### 竞品官网 / 运营页面

- **元素**：「新闻动态」「产品发布」等区域
- **频率**：每 6～24 小时
- **通知**：邮件 / 企业微信

::: warning 登录态
后台看板、内网页面请配合 [cookie plus](../features/cookie-plus)。
:::

---

## 🔌 HTTP 请求

### 接口可用性 / 健康检查

- **配置**：GET `https://api.example.com/health`
- **规则**：响应码等于 200；或响应体包含 `"ok"`
- **频率**：每 1～5 分钟
- **通知**：Webhook 接入告警系统，或钉钉

### 开放 API / 支付回调巡检

- **配置**：POST + 请求头 / Body（按需）
- **规则**：响应码不等于 200、响应内容变化、超时
- **频率**：每 5～15 分钟

---

## 📡 RSS 订阅

### 博客 / 播客 / 资讯源

- **字段**：`title`、`link`、`pubDate`
- **规则**：内容变化或包含关键词
- **频率**：每 1～2 小时
- **通知**：飞书 / pushplus

---

## 📅 域名与证书

### 主域名续费

- **类型**：域名到期时间
- **规则**：距到期不足 30 天
- **频率**：每天
- **通知**：邮件 + 企业微信

### HTTPS 证书临期

- **类型**：网站证书
- **规则**：距到期不足 14 天
- **频率**：每天
- **示例**：生产官网、内网网关 TLS

---

## 📶 Ping 检测

### 服务器 / NAS / 路由器

- **目标**：IP 或域名（可从 URL 解析主机）
- **规则**：Ping 不可达；平均延迟大于阈值
- **频率**：每 1～5 分钟
- **通知**：本地 + Webhook

客户端内可先 **测试 Ping** 再保存任务。

---

## 🎫 票务 / 报名（网站类）

- **元素**：「立即购买」按钮状态、库存文案
- **频率**：每 1～5 分钟
- **通知**：pushplus + 本地（提示音）

---

## 选择技巧（网站任务）

1. 选 **稳定区域**，避开轮播、推荐位
2. **标题 + 时间** 组合比单标题更准
3. 保存前 **立即执行** 验证
4. 登录页先完成 cookie plus 再监控

## 不太适合的场景

- ❌ 纯图片、无文本区域（暂无图像 OCR 对比）
- ❌ 「猜你喜欢」等随机推荐位
- ❌ 强反爬且无合法 Cookie 的页面
- ❌ 需人脸等复杂验证的登录流程

更多类型说明见 [监控任务类型](/features/task-types)。
