# 钉钉 / 企业微信 / 飞书机器人

这三种渠道都是基于「群机器人 Webhook」推送，配置流程基本一致：

1. 在 IM 群里创建一个机器人，得到一个 **Webhook URL**
2. 把 URL 填到网页侦探的渠道配置里
3. 测试发送

::: tip 谁能收到
机器人发送的消息会被群里 **所有成员** 看到，适合团队共享监控提醒。
:::

## 钉钉机器人

### 1. 在钉钉群创建机器人

1. 打开钉钉群 → 右上角 ⚙ → **群助手 → 添加机器人 → 自定义**
2. 安全设置勾选「**自定义关键词**」或「**加签**」
   - 自定义关键词推荐填：`网页侦探`、`变化提醒`
3. 复制 **Webhook URL**

### 2. 在网页侦探中配置

「我的 → 通知渠道 → 添加 → 钉钉机器人」：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 渠道名称 | ✅ | 如「钉钉运营群」 |
| Webhook URL | ✅ | 钉钉机器人完整 URL |
| 加签密钥 | - | 若安全设置选了「加签」则必填 |
| @ 谁 | - | 手机号列表，逗号分隔；填 `all` 则 @所有人 |

### 3. 测试

点击 <kbd>测试</kbd>，群里收到「网页侦探测试」即配置成功。

::: warning 安全设置
钉钉机器人必须至少配置一种安全设置，否则会拒绝 Webhook 请求。
推荐用「自定义关键词」最简单，关键词包含 `网页侦探` 即可。
:::

## 企业微信机器人

### 1. 在企业微信群创建机器人

1. 打开企业微信群 → 右上角 `⋯` → **添加群机器人**
2. 新建机器人，自定义名称和头像
3. 复制 **Webhook URL**

### 2. 在网页侦探中配置

「我的 → 通知渠道 → 添加 → 企业微信机器人」：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 渠道名称 | ✅ | 如「企微研发群」 |
| Webhook URL | ✅ | 完整的企微 webhook url |
| @ 谁 | - | 企业微信用户 ID 或手机号 |

### 3. 测试

点击 <kbd>测试</kbd>，群里收到测试消息即可。

::: tip 注意
企业微信机器人没有「关键词」「加签」等安全要求，
但有 **每分钟 20 条** 的发送频率限制，请勿密集触发。
:::

## 飞书机器人

### 1. 在飞书群创建机器人

1. 打开飞书群 → 右上角 `⋯` → **设置 → 群机器人 → 添加机器人**
2. 选择「自定义机器人」
3. 安全设置勾选「**自定义关键词**」或「**签名校验**」
4. 复制 **Webhook URL**

### 2. 在网页侦探中配置

「我的 → 通知渠道 → 添加 → 飞书机器人」：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 渠道名称 | ✅ | 如「飞书产品群」 |
| Webhook URL | ✅ | 完整的飞书 webhook url |
| 签名 secret | - | 若选了「签名校验」则必填 |
| @ 谁 | - | 飞书 open_id 列表 |

### 3. 测试

点击 <kbd>测试</kbd>，群里收到测试消息即配置成功。

---

## 消息样式

三种 IM 渠道都使用各自的 **富文本卡片** 样式：

- 标题：任务名 + 「检测到变化」
- 正文：变化摘要（最多 5 行）
- 行动按钮：「查看网页」「查看执行记录」

## 失败排查

| 现象 | 排查方向 |
| --- | --- |
| 测试失败：`关键词不匹配` | 钉钉 / 飞书的自定义关键词需要包含「网页侦探」 |
| 测试失败：`签名错误` | 是否填了加签密钥 / 签名 secret |
| 测试失败：`webhook 已禁用` | 群机器人是否被群主移除 |
| 偶发失败 | 是否触发限流（钉钉 5s 内最多 20 条） |
