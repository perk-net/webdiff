# 任务管理

**任务** 是网页侦探的核心单元：一个任务 = 一种监控类型 + 监控目标 + 触发规则 + 执行频率 + 通知配置。

七类任务类型（网站、HTTP、RSS、域名、证书、Ping、自定义脚本）的字段与规则各不相同，请先阅读 [监控任务类型](./task-types.md)。自定义脚本详见 [自定义脚本](./task-script.md)。

## 任务的生命周期

```
新建（选类型）→ 配置目标与规则 → 启用 → 调度执行 → 满足规则 → 触发通知
                        ↓
                   停用 / 删除
```

## 通用字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 任务名称 | ✅ | 用于辨识，最长 50 字 |
| 任务类型 | ✅ | 决定配置向导与执行方式 |
| 监控目标 | ✅ | 网站 / HTTP / RSS / Ping 为 URL；域名为域名；证书为站点 URL |
| Cron / 频率 | ✅ | 详见 [定时与执行频率](./cron.md) |
| 触发规则 | ✅ | 因类型而异，见 [监控任务类型](./task-types.md) |
| 运行客户端 | ✅ | 决定任务在哪台客户端执行，见 [运行客户端](../client/run-client.md) |
| 通知渠道 | ❌ | 可绑定 0~N 个 |
| Cookie（部分类型） | ❌ | 网站 / HTTP 可使用 cookie plus |
| 启用状态 | - | `运行` / `停用` |

网站类任务另有 **监控元素**（XPath / CSS），至少 1 个，最多 20 个。

## 新建任务

主界面 <kbd>+ 新增任务</kbd> 或快捷键 `Ctrl + N`（macOS 为 `Cmd + N`）。

1. 选择 **任务类型**
2. 按向导逐步填写（步骤因类型而异）
3. 保存后默认 **运行**，加入调度

## 编辑任务

任务列表点 <kbd>编辑</kbd> 可修改任意配置，**保存后立即生效**。

::: warning 网站任务：选择器变更
修改监控元素后，对比基线会重置，首次执行通常不会产生「变化」通知。
:::

::: warning 规则或目标大幅变更
建议改完后 **立即执行** 一次，确认规则与通知符合预期。
:::

## 启用 / 停用

- **运行**：按 Cron 自动执行
- **停用**：不再调度，历史记录保留

::: tip 配额提示
免费用户最多 **10 个运行中** 任务，超出需 [升级会员](../guide/membership.md)。停用任务不计入配额。
:::

## 立即执行

用于验证配置、测试通知，结果写入 [执行记录](./records.md)，与定时执行同等记录。

## 删除任务

<kbd>删除</kbd> 经二次确认后永久删除，含执行记录、消息记录与网站类 HTML 快照。**不可恢复**。

## 搜索与筛选

- 按 **名称 / 目标** 搜索
- 按 **状态**（运行 / 停用）筛选
- 按 **运行客户端**（单节点 / 全节点 / 具体设备）筛选，详见 [运行客户端](../client/run-client.md)
- 按 **创建时间** 排序（默认倒序）

## 导入 / 导出

当前版本暂不支持任务 JSON 导入导出，可通过备份 `%APPDATA%\perk-webdiff\`（或 macOS 对应目录）迁移整机数据。
