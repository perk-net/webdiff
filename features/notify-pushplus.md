# pushplus

[pushplus](https://www.pushplus.plus/) 是一个免费的微信推送服务，
可以把任意消息推送到你的微信。配置后，网页侦探的通知会直达微信。

## 准备工作

1. 打开 [https://www.pushplus.plus/](https://www.pushplus.plus/)
2. 使用微信扫码登录
3. 进入「一对一推送」或「一对多推送」
4. 复制你的 **token**（一长串字符）

## 在网页侦探中配置

「我的 → 通知渠道 → 添加渠道」，类型选 **pushplus**：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 渠道名称 | ✅ | 自定义，如「我的微信」 |
| token | ✅ | 上一步复制的 pushplus token |
| 模板类型 | - | 默认 `html`，可选 `markdown` |
| 群组编码 | - | 群发时填，留空为一对一 |

点击 <kbd>测试</kbd>，微信收到一条「网页侦探测试」即配置成功。

## 群发到群组

pushplus 支持把消息推送到一个群组（多人都能收到）：

1. 在 pushplus 平台「群组」页创建群组，得到一个 `topic` 字符串
2. 告诉其他成员扫码加入该群组
3. 在网页侦探的渠道配置里，把 `topic` 填到「群组编码」字段

::: tip 适合场景
- 团队共享一个监控任务
- 家人朋友共同接收抢购提醒
:::

## 消息样式

pushplus 通知在微信里以服务号消息形式呈现，包含：

- 标题：任务名 + 「内容已变化」
- 正文：变化摘要 + 任务网址链接
- 点击：跳转到该任务的网址

## 限流注意

pushplus 免费版有每日推送数量限制（具体见 pushplus 官网）。
高频任务建议：

- 多个任务复用 1 个 token，避免每个任务都开新 token
- 重要任务用 pushplus，次要任务用本地通知 / 群机器人
- 必要时可在 pushplus 升级会员获取更高额度

## token 安全

- token 仅保存在 **本机** SQLite 中
- 不会上传到网页侦探的任何服务器
- 如果不慎泄露 token，请到 pushplus 平台 **重置** 即可

## 失败排查

| 现象 | 排查方向 |
| --- | --- |
| 测试失败：`token 错误` | 复制时是否带了空格 / 多余字符 |
| 测试成功，但任务通知收不到 | 任务是否真的发生了变化（看执行记录） |
| 微信端没消息 | pushplus 服务号是否仍关注，是否被限流 |
| 偶发失败 | pushplus 服务抖动，重试通常能解决 |
