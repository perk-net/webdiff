# RSS 订阅

订阅 RSS/Atom 源，可指定 `title`、`link`、`description`、`pubDate` 等字段，有新条目或字段变化时通知。

## 适用场景

- 博客、资讯站更新提醒
- 产品更新日志 / Changelog 订阅
- 聚合源中特定字段变化监控

## 创建步骤



### 选择 RSS 订阅类型

新建任务并选择「RSS 订阅」，用于跟踪 Feed 源的新条目或字段变化。

![选择 RSS 订阅](../public/features/rss/step1.png)



### 填写 RSS 地址

输入 RSS/Atom 链接，客户端会拉取并解析条目列表。

![填写 RSS 地址](../public/features/rss/step2.png)



### 选择监控字段

勾选要关注的字段（如 title、link、description、pubDate）。

![选择监控字段](../public/features/rss/step3.png)



### 规则、频率与通知

配置关键词/内容变化等规则与执行频率，绑定通知后保存。

![规则频率与通知](../public/features/rss/step4.png)



## 触发规则

内容变化、包含/不包含关键词、正则、超时
